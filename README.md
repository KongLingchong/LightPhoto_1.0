## how to use?
1. run the code
2. click "开始修图"(start to fix),chooce your photo that need to fix.
3. then,you will see a window,you can click any button to fix your photo.
   Tips:
   效果 means effects.
   调色 means color.
   降噪 means make the photo clear.
   完成修图 means finish.
